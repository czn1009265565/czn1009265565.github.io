## 交易数据

### 环境准备
Python 的版本需要在 `Python 3.8` 以上，推荐使用 `Python 3.10.x` 及以上版本

```shell
pip install akshare -i http://mirrors.aliyun.com/pypi/simple/ --trusted-host=mirrors.aliyun.com  --upgrade
```
