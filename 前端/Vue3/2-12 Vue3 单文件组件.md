# Vue3 单文件组件

单文件组件（Single-File Component，简称 SFC） 是构建用户界面的核心单元。
它将一个组件的 HTML（结构）、JavaScript（逻辑） 和 CSS（样式） 封装在同一个 `.vue` 文件中

##  SFC 的标准骨架
一个标准的 Vue 3 单文件组件（.vue 文件）由三个顶级代码块组成

```vue
<!-- 1. 模板：定义组件的 HTML 结构 -->
<template>
  <div class="my-component">
    <h1>{{ title }}</h1>
    <button @click="handleClick">Click me</button>
  </div>
</template>

<!-- 2. 逻辑：编写组件的 JavaScript/TypeScript 代码 -->
<script setup>
import { ref } from 'vue'

const title = ref('Hello Vue 3!')

function handleClick() {
  console.log('Button clicked!')
}
</script>

<!-- 3. 样式：定义组件的 CSS 样式 -->
<style scoped>
.my-component {
  padding: 20px;
  border: 1px solid #ccc;
}
</style>
```

## 使用组件

### 定义子组件
UserCard.vue
```vue
<template>
  <div class="user-card" :class="{ 'is-vip': isVip }">
    <h2 class="username">{{ name }}</h2>
    <p class="score">Score: {{ score }}</p>
    
    <button @click="addScore" class="btn">Add Score</button>
  </div>
</template>

<script setup>
import { ref, computed } from 'vue'

const name = ref('Alice')
const score = ref(0)

// 计算属性
const isVip = computed(() => score.value >= 100)

// 内部方法
function addScore() {
  score.value += 10
}
</script>

<style scoped>
</style>
```

### 父组件使用

App.vue

```vue
<template>
  <div id="app">
    <h1>My Users</h1>
    <UserCard />
    <UserCard />
  </div>
</template>

<script setup>
// 仅需导入，Vue 3 会自动将其注册为同名标签
import UserCard from './components/UserCard.vue'
</script>
```
