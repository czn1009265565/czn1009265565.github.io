# Vue3 安装

## 独立版本
官网地址 `https://unpkg.com/vue@3/dist/vue.global.js`, 下载后使用 `<script>` 标签引入即可

## CDN
- cdnjs: `https://cdnjs.cloudflare.com/ajax/libs/vue/3.0.5/vue.global.js`
- Staticfile CDN: `https://cdn.staticfile.net/vue/3.0.5/vue.global.js`

示例

```html
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>Vue 测试实例</title>
<script src="https://cdnjs.cloudflare.com/ajax/libs/vue/3.0.5/vue.global.js"></script>
</head>
<body>
<div id="hello-vue" class="demo">
  {{ message }}
</div>

<script>
const HelloVueApp = {
  data() {
    return {
      message: 'Hello Vue!!'
    }
  }
}

Vue.createApp(HelloVueApp).mount('#hello-vue')
</script>
</body>
</html>
```

## npm
由于 npm 安装速度慢,国内环境建议配置镜像源

```shell
npm -v
# 11.9.0

# 配置镜像源
npm config set registry https://registry.npmmirror.com
```

这里简单比对下不同的项目创建命令:
- `npm init vue@latest`: 创建一个功能完整、官方推荐的 Vue 项目模板（基于 Vite）。在初始化时询问你是否需要 Router、Pinia、TypeScript等，并自动配置好
- `npm create vite@latest`: 创建一个极简、轻量的 Vite + Vue 项目。你需要自己手动安装路由、状态管理等库
- `vue ui` 在浏览器中打开图形界面来创建项目 (已弃用)
- `npm init vite-app` Vite 早期的脚手架命令 (已弃用)，被 `npm create vite` 取代
