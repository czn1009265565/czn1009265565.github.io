# Vue3 声明式渲染

声明式渲染（Declarative Rendering）是指通过数据驱动视图的更新，而不是直接操作 DOM。

Vue3 的声明式渲染是一种基于模板的渲染方式，
它允许开发者通过简洁的模板语法来描述页面的结构和数据绑定关系，而不需要直接操作 DOM。

## 指令

Vue 指令是带有前缀 v- 的特殊 HTML 属性，它赋予 HTML 标签额外的功能

### 常用指令

- v-bind	用于将 Vue 实例的数据绑定到 HTML 元素的属性上。
- v-if	用于根据表达式的值来条件性地渲染元素或组件。
- v-show	v-show 是 Vue.js 提供的一种指令，用于根据表达式的值来条件性地显示或隐藏元素。
- v-for	用于根据数组或对象的属性值来循环渲染元素或组件。
- v-on	用于在 HTML 元素上绑定事件监听器，使其能够触发 Vue 实例中的方法或函数。
- v-model	用于在表单控件和 Vue 实例的数据之间创建双向数据绑定

## 模板语法

```vue
<template>
  <div id="app">
    <!-- 1. 文本插值 -->
    <h1>{{ title }}</h1>

    <!-- 2. 属性绑定 -->
    <img v-bind:src="logoUrl" alt="Logo1" />
    <img :src="logoUrl" alt="Logo2" />
    <!-- 双向数据绑定 -->
    <div>
      <input v-model="message" placeholder="输入一些文本" />
      <p>你输入了：{{ message }}</p>
    </div>

    <!-- 3. 条件渲染 -->
    <p v-if="isLoggedIn">欢迎回来, {{ username }}!</p>
    <p v-else>请登录</p>

    <!-- 4. 列表渲染 -->
    <ul>
      <li v-for="todo in todos" :key="todo.id">
        {{ todo.text }}
      </li>
    </ul>

    <!-- 5. 事件处理 -->
    <button v-on:click="addTodo">添加待办</button>
    <button @click="addTodo">添加待办</button>
  </div>
</template>
```

## 实现方式

### 单文件 HTML 实现
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>Vue 3 声明式渲染示例</title>
    <!-- 1. 引入 Vue 3 全局构建版本 -->
    <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
</head>
<body>

    <!-- 2. 挂载点：Vue 将接管这个 div -->
    <div id="app">
        <!-- 模板部分：与 .vue 文件中的 <template> 内容完全一致 -->
        <p>Count is: {{ count }}</p>
        <button @click="count++">增加</button>
    </div>

    <script>
        // 3. 从全局 Vue 对象中解构出 createApp 和 ref
        const { createApp, ref } = Vue;

        // 4. 创建应用实例
        const app = createApp({
            // setup 函数对应 <script setup> 的逻辑部分
            setup() {
                //定义响应式数据
                const count = ref(0);
                // 【关键步骤】必须将变量返回，模板才能使用
                return {
                    count
                };
            }
        });

        // 5. 挂载到 #app 元素
        app.mount('#app');
    </script>
</body>
</html>
```

### 工程化实现
```vue
<template>
  <!-- 你只声明：这里显示 count 的值 -->
  <p>Count is: {{ count }}</p>
  <button @click="count++">增加</button>
</template>

<script setup>
import { ref } from 'vue';
const count = ref(0);
</script>
```

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>Vue 3 Global Build 示例</title>
  <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>
</head>
<body>

<div id="app">
  <!-- 1. 文本插值 -->
  <h1>{{ title }}</h1>

  <!-- 2. 属性绑定 -->
  <img v-bind:src="logoUrl" alt="Logo1" />
  <img :src="logoUrl" alt="Logo2" />
  <!-- 双向数据绑定 -->
  <div>
      <input v-model="message" placeholder="输入一些文本" />
      <p>你输入了：{{ message }}</p>
  </div>

  <!-- 3. 条件渲染 -->
  <p v-if="isLoggedIn">欢迎回来, {{ username }}!</p>
  <p v-else>请登录</p>

  <!-- 4. 列表渲染 -->
  <ul>
    <li v-for="todo in todos" :key="todo.id">
      {{ todo.text }}
    </li>
  </ul>

  <!-- 5. 事件处理 -->
  <button v-on:click="addTodo">添加待办</button>
  <button @click="addTodo">添加待办</button>
</div>


<script>
  const { createApp, ref, defineComponent } = Vue;

  const app = createApp({

    setup() {
      // 定义响应式数据
      const title = ref('我的待办列表');
      const logoUrl = ref('https://vuejs.org/logo.png');
      const message = ref('');
      const isLoggedIn = ref(true);
      const username = ref('Vue User');

      const todos = ref([
        { id: 1, text: '学习 Vue 3' },
        { id: 2, text: '写代码' }
      ]);

      // 定义处理函数
      function addTodo() {
        // 只需修改数据，DOM 会自动更新
        todos.value.push({
          id: Date.now(),
          text: '新待办事项'
        });
      }

      // 必须 return 出去才能在模板中使用
      return {
        title,
        logoUrl, 
        message,
        isLoggedIn,
        username,
        todos,
        addTodo
      };
    }
  });
  app.mount('#app');
</script>
</body>
</html>
```
