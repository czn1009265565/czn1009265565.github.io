# Vue3 表单

处理表单的核心是 v-model 指令。它实现了表单输入元素和应用状态之间的双向数据绑定

v-model指令:
- 绑定值: 将数据（State）显示在视图（View）上。
- 监听更新: 当用户在视图上修改内容时，自动更新数据。

它是 v-bind:value 和 v-on:input（或其他事件）的语法糖

## 基础用法

### 单行文本

```vue
<template>
  <input v-model="message" placeholder="请输入内容" />
  <p>消息: {{ message }}</p>
</template>

<script setup>
import { ref } from 'vue'
const message = ref('')
</script>
```

### 多行文本

```vue
<template>
  <textarea v-model="multilineText" placeholder="多行输入"></textarea>
  <p style="white-space: pre-line;">{{ multilineText }}</p>
</template>

<script setup>
import { ref } from 'vue'
const multilineText = ref('')
</script>
```

### 复选框
单复选框
```vue
<template>
  <input type="checkbox" id="checkbox" v-model="checked" />
  <label for="checkbox">{{ checked }}</label> <!-- true / false -->
</template>

<script setup>
  import { ref } from 'vue'
  const checked = ref(false)
</script>
```

多复选框

```vue
<template>
  <div>
    <input type="checkbox" id="jack" value="Jack" v-model="checkedNames" />
    <label for="jack">Jack</label>

    <input type="checkbox" id="john" value="John" v-model="checkedNames" />
    <label for="john">John</label>

    <input type="checkbox" id="mike" value="Mike" v-model="checkedNames" />
    <label for="mike">Mike</label>
  </div>
  <p>选中的人: {{ checkedNames }}</p> <!-- ['Jack', 'John'] -->
</template>

<script setup>
  import { ref } from 'vue'
  const checkedNames = ref([])
</script>
```
注意: 绑定的变量必须初始化为数组 []

## 值绑定

对于单选、复选和选择框，v-model 绑定的值通常是一个字符串。但有时我们需要绑定动态值（如对象或数字）

```vue
<template>
  <div>
    <h3>单选按钮组示例</h3>

    <!-- 第一个单选按钮 -->
    <label>
      <input type="radio" v-model="selectedItem" :value="itemA" />
      选项A
    </label>

    <!-- 第二个单选按钮 -->
    <label>
      <input type="radio" v-model="selectedItem" :value="itemB" />
      选项B
    </label>

    <!-- 第三个单选按钮（用于取消选择） -->
    <label>
      <input type="radio" v-model="selectedItem" :value="null" />
      取消选择
    </label>

    <div style="margin-top: 20px;">
      <p>当前选中的值: {{ selectedItem }}</p>
      <p v-if="selectedItem">选中对象的name: {{ selectedItem?.name }}</p>
    </div>
  </div>
</template>

<script setup>
  import { ref } from 'vue'

  const itemA = ref({ id: 1, name: '对象A' })
  const itemB = ref({ id: 2, name: '对象B' })
  const selectedItem = ref(null)  // 初始未选择
</script>
```

## 修饰符

### `.lazy`

默认情况下，v-model 在每次 input 事件触发后同步数据。添加 .lazy 后，改为在 change 事件后同步（即失去焦点或按回车时）

```vue
<!-- 在 "change" 而非 "input" 时更新 -->
<input v-model.lazy="msg" />
```

### `.number`
自动将用户的输入值转为数值类型。如果无法被 parseFloat() 解析，则返回原始字符串
```vue
<input v-model.number="age" type="number" />
```

### `.trim`

自动过滤用户输入的首尾空白字符

```vue
<input v-model.trim="msg" />
```
