# Vue3 事件处理

Vue 3 的事件处理机制（v-on / @）主要解决了前端开发中交互逻辑与 DOM 操作耦合、代码可维护性差以及跨组件通信困难这三大核心问题。

## 监听原生 DOM 事件

### 基本语法
使用 `@event="handler"` 绑定方法
```vue
<template>
  <!-- 点击按钮触发 greet 方法 -->
  <button @click="greet">打招呼</button>
  
  <!-- 也可以直接写内联表达式（简单逻辑） -->
  <button @click="count++">计数: {{ count }}</button>
</template>

<script setup>
import { ref } from 'vue'

const count = ref(0)

const greet = (event) => {
  console.log('Hello!', event.target) // event 是原生 DOM 事件对象
}
</script>
```

### 传递参数
如果需要传递额外参数，同时保留原生事件对象 `$event`

```vue
<template>
  <button @click="say('hello', $event)">Say Hello</button>
</template>

<script setup>
const say = (message, event) => {
  console.log(message)
  console.log(event)
}
</script>
```

## 事件修饰符

Vue 提供了后缀点表示法来调用 `event.preventDefault()` 或 `event.stopPropagation()`，让代码更简洁

| 修饰符      | 	作用                           | 	等价原生 JS                                   |
|----------|-------------------------------|--------------------------------------------|
| .stop    | 	阻止事件冒泡                       | 	event.stopPropagation()                   |
| .prevent | 	阻止默认行为                       | 	event.preventDefault()                    |
| .capture | 	使用捕获模式添加事件                   | 	addEventListener(..., true)               |
| .self    | 	只有当事件是从侦听器绑定的元素本身触发时才触发回调    | 	if (event.target === event.currentTarget) |
| .once    | 	事件只触发一次                      | 	removeEventListener after first call      |
| .passive | 	提升滚动性能（主要用于 touch/wheel 事件）	 | { passive: true }                          |

```vue
<template>
  <!-- 阻止表单提交刷新页面 -->
  <form @submit.prevent="hi">提交</form>

  <div @click="parentHandler">
    <!-- 阻止点击事件冒泡到父元素 -->
    <button @click.stop="childHandler">ButtonA</button>
    <!-- 不阻止点击事件冒泡到父元素 -->
    <button @click="childHandler">ButtonB</button>
  </div>

  <!-- 组合修饰符：先 prevent 再 stop -->
  <a @click.prevent.stop="hi">链接</a>

  <!-- 只触发一次 -->
  <button @click.once="hi">只能点一次</button>

</template>

<script setup>
const parentHandler = () => {
  console.log("parent");
}
const childHandler = () => {
  console.log("child");
}
const hi = () => {
  console.log("Hi~");
}
</script>
```

### DOM 事件类型

| 事件类别      | 事件类型           | 语法示例                               | 描述           |
|-----------|----------------|------------------------------------|--------------|
| **鼠标事件**  | `click`        | `@click="handleClick"`             | 点击事件         |
|           | `dblclick`     | `@dblclick="handleDoubleClick"`    | 双击事件         |
|           | `mousedown`    | `@mousedown="handleMouseDown"`     | 鼠标按下         |
|           | `mouseup`      | `@mouseup="handleMouseUp"`         | 鼠标释放         |
|           | `mouseenter`   | `@mouseenter="handleMouseEnter"`   | 鼠标进入元素       |
|           | `mouseleave`   | `@mouseleave="handleMouseLeave"`   | 鼠标离开元素       |
|           | `mousemove`    | `@mousemove="handleMouseMove"`     | 鼠标移动         |
|           | `mouseover`    | `@mouseover="handleMouseOver"`     | 鼠标悬停（冒泡）     |
|           | `mouseout`     | `@mouseout="handleMouseOut"`       | 鼠标离开（冒泡）     |
| **键盘事件**  | `keydown`      | `@keydown="handleKeyDown"`         | 按键按下         |
|           | `keyup`        | `@keyup="handleKeyUp"`             | 按键释放         |
|           | `keypress`     | `@keypress="handleKeyPress"`       | 按键输入（字符键）    |
|           | `keyup.enter`  | `@keyup.enter="handleEnter"`       | 回车键          |
|           | `keyup.esc`    | `@keyup.esc="handleEscape"`        | ESC键         |
|           | `keyup.space`  | `@keyup.space="handleSpace"`       | 空格键          |
|           | `keyup.tab`    | `@keyup.tab="handleTab"`           | Tab键         |
| **表单事件**  | `input`        | `@input="handleInput"`             | 输入事件         |
|           | `change`       | `@change="handleChange"`           | 内容改变         |
|           | `focus`        | `@focus="handleFocus"`             | 获得焦点         |
|           | `blur`         | `@blur="handleBlur"`               | 失去焦点         |
|           | `submit`       | `@submit.prevent="handleSubmit"`   | 表单提交         |
| **窗口事件**  | `resize`       | `@resize="handleResize"`           | 窗口大小改变       |
|           | `scroll`       | `@scroll="handleScroll"`           | 滚动事件         |
|           | `load`         | `@load="handleLoad"`               | 加载完成         |
|           | `unload`       | `@unload="handleUnload"`           | 卸载事件         |
| **剪贴板事件** | `copy`         | `@copy="handleCopy"`               | 复制事件         |
|           | `cut`          | `@cut="handleCut"`                 | 剪切事件         |
|           | `paste`        | `@paste="handlePaste"`             | 粘贴事件         |
| **拖放事件**  | `dragstart`    | `@dragstart="handleDragStart"`     | 开始拖拽         |
|           | `drag`         | `@drag="handleDrag"`               | 拖拽中          |
|           | `dragend`      | `@dragend="handleDragEnd"`         | 拖拽结束         |
|           | `dragover`     | `@dragover="handleDragOver"`       | 拖拽经过         |
|           | `dragenter`    | `@dragenter="handleDragEnter"`     | 拖拽进入         |
|           | `dragleave`    | `@dragleave="handleDragLeave"`     | 拖拽离开         |
|           | `drop`         | `@drop="handleDrop"`               | 放置事件         |
| **媒体事件**  | `loadeddata`   | `@loadeddata="handleLoadedData"`   | 媒体数据加载       |
|           | `canplay`      | `@canplay="handleCanPlay"`         | 可以播放         |
|           | `play`         | `@play="handlePlay"`               | 开始播放         |
|           | `pause`        | `@pause="handlePause"`             | 暂停播放         |
|           | `ended`        | `@ended="handleEnded"`             | 播放结束         |
|           | `timeupdate`   | `@timeupdate="handleTimeUpdate"`   | 时间更新         |
| **触摸事件**  | `touchstart`   | `@touchstart="handleTouchStart"`   | 触摸开始         |
|           | `touchmove`    | `@touchmove="handleTouchMove"`     | 触摸移动         |
|           | `touchend`     | `@touchend="handleTouchEnd"`       | 触摸结束         |
|           | `touchcancel`  | `@touchcancel="handleTouchCancel"` | 触摸取消         |
| **自定义事件** | `custom-event` | `@custom-event="handleCustom"`     | 自定义事件        |
|           | `update:model` | `@update:model="handleUpdate"`     | v-model 更新事件 |
