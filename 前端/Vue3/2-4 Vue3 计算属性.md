# Vue3 计算属性

计算属性用于根据其他数据的变化动态计算衍生出来的属性值，而且具有缓存机制，
只有相关依赖发生变化时才会重新计算。

计算属性关键词: computed

## 单 HTML 文件实现
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>Vue 测试实例</title>
    <script src="https://cdn.staticfile.org/vue/3.0.5/vue.global.js"></script>
</head>
<body>
<div id="app">
    <p>商品名称：{{ productName }}</p>
    <p>商品价格：{{ formattedPrice }}</p>
    <button @click="increasePrice">增加价格</button>
</div>

<script>
    const {ref, computed} = Vue;

    const app = {
        setup() {
            const state = ref({
                name: '手机',
                price: 2000
            });

            const productName = computed(() => {
                return `优惠 ${state.value.name}`;
            });

            const formattedPrice = computed(() => {
                return `￥${state.value.price.toFixed(2)}`;
            });

            const increasePrice = () => {
                state.value.price += 100;
            };
            return {state, productName, formattedPrice, increasePrice}
        }
    }

    Vue.createApp(app).mount('#app')
</script>
</body>
</html>
```

    return `优惠 ${state.value.name}`;
  });
