# Vue3 样式绑定

样式绑定主要通过 v-bind（简写为 :）指令来实现。Vue 提供了两种主要方式：绑定 Class 和 绑定 Style

优先使用 Class 绑定：尽量通过切换 Class 来改变样式，而不是直接操作 :style。这样更符合关注点分离原则，也更容易维护。

## Class 属性绑定

### 对象语法
最常用

```vue
<template>
  <!-- isActive 为 true 时，添加 'active' 类 -->
  <div :class="{ active: isActive, 'text-danger': hasError }">
    Hello World
  </div>
</template>

<script setup>
import { ref } from 'vue'

const isActive = ref(true)
const hasError = ref(false)
</script>
```

### 数组语法

```vue
<template>
  <div :class="[baseClass, activeClass]">
    Hello World
  </div>
</template>

<script setup>
import { ref } from 'vue'

const baseClass = ref('container')
const activeClass = ref('active')
</script>
```

### 三元表达式

```vue
<template>
  <div :class="[isActive ? 'active-class' : '', errorClass]">
    Hello World
  </div>
</template>

<script setup>
  import { ref } from 'vue'

  const isActive = ref(true)
  const errorClass = ref('error')
</script>
```

## 绑定内联样式

### 对象语法
```vue
<template>
  <div :style="{ color: activeColor, fontSize: fontSize + 'px' }">
    Hello World
  </div>
</template>

<script setup>
import { ref } from 'vue'

const activeColor = ref('red')
const fontSize = ref(30)
</script>
```

### 绑定样式对象

```vue
<template>
  <div :style="styleObject">Hello World</div>
</template>

<script setup>
import { ref } from 'vue'

const styleObject = ref({
  color: 'blue',
  fontSize: '13px'
})
</script>
```

### 多重样式

支持多个样式对象，后面的属性会覆盖前面

```vue
<template>
  <div :style="[baseStyles, overridingStyles]">Hello World</div>
</template>

<script setup>
import { ref } from 'vue'

const baseStyles = ref({
  color: 'blue',
  fontSize: '20px'
})

const overridingStyles = ref({
  color: 'green' // 最终颜色为 green
})
</script>
```

## 在 CSS 中使用 v-bind()
这是 Vue 3 <script setup> 的一个强大功能，允许你直接在 <style> 标签中引用 JS 变量

```vue
<template>
  <div class="text">Hello World</div>
  <button @click="changeColor">改变颜色</button>
</template>

<script setup>
import { ref } from 'vue'

const themeColor = ref('red')
const fontSize = ref('16px')

const changeColor = () => {
  themeColor.value = themeColor.value === 'red' ? 'blue' : 'red'
}
</script>

<style scoped>
.text {
  /* 直接绑定 JS 变量 */
  color: v-bind(themeColor);
  font-size: v-bind(fontSize); /* 也支持其他变量 */
}
</style>
```
