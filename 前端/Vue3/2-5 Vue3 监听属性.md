# Vue3 监听属性

监听响应式数据的变化主要通过 watch 和 watchEffect 两个 API 实现，
用于在数据变化时执行异步操作、DOM 更新或复杂逻辑。

在大多数业务逻辑中（如 API 请求、表单处理），watch 更常用且更安全。

## Watch
watch 的作用是用于监测响应式属性的变化，并在属性发生改变时执行自定义的逻辑。

`watch(source, callback, [options])`

- source 监听源：可以是 ref、reactive、getter 函数 或数组
- callback 回调函数：(newValue, oldValue) => { ... }
- options 配置项：immediate, deep, flush 等


### 监听单属性

```vue
<script setup>
  import { ref, watch } from 'vue'

  const count = ref(0);

  watch(count, (newValue, oldValue) => {
    console.log(`count 从 ${oldValue} 变为 ${newValue}`)
  })
</script>
```

### 监听多属性

```vue
<script setup>
    const name = ref('Alice')
    const age = ref(25)
    
    // 回调中的参数也是数组
    watch([name, age], ([newName, newAge], [oldName, oldAge]) => {
      console.log(`name: ${oldName} -> ${newName}`)
      console.log(`age: ${oldAge} -> ${newAge}`)
    })
</script>
```

### 配置项

```javascript
watch(source, callback, {
    immediate: true, // 立即执行一次回调
    deep: true,      // 深度监听（针对 ref 包裹的对象）
    flush: 'pre'     // 'pre'(默认, DOM更新前), 'post'(DOM更新后), 'sync'(同步)
})
```

### 搜索防抖
```vue
<script setup>
import { ref, watch } from 'vue'

const keyword = ref('');
const results = ref([]);
// 用于防抖定时器
let timer = null;

// 监听 keyword 变化
watch(keyword, (newVal, oldValue) => {
  // 1. 清除上一次未执行的请求（防抖核心）
  clearTimeout(timer)

  // 2. 如果为空，清空结果并停止
  if (!newVal.trim()) {
    results.value = []
    return
  }

  // 3. 设置延迟执行（模拟防抖 500ms）
  timer = setTimeout(async () => {
    console.log(`正在搜索: ${newVal}`)
    // 模拟 API 请求
    const res = await fetch(`https://jsonplaceholder.typicode.com/posts?title_like=${newVal}`)
    const data = await res.json()
    results.value = data.slice(0, 5) // 只取前5条
  }, 500)
})
</script>

<template>
  <div style="padding: 20px;">
    <input v-model="keyword" placeholder="输入关键词搜索 (如: qui, sunt)" />

    <ul>
      <li v-for="item in results" :key="item.id">
        {{ item.title }}
      </li>
    </ul>

    <p v-if="!results.length && keyword" style="color: #999;">加载中...</p>
  </div>
</template>
```

## watchEffect
特点：自动追踪依赖，立即执行，代码更简洁，但无法获取旧值

```vue
<script setup>
  import { ref, watchEffect } from 'vue'

  const count = ref(0);
  const name = ref('Alice');

  watchEffect(() => {
    // 自动追踪 count 和 name
    console.log(`Count: ${count.value}, Name: ${name.value}`)
  });
</script>
```
