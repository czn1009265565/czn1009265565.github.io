```javascript
import { createApp } from 'vue'

const app = createApp({

})
```

## 响应式数据

ref 和 reactive 是创建响应式数据的两个核心 API, 官方更推荐统一使用 ref

### ref
ref 接受一个内部值并返回一个响应式的 ref 对象。这个对象只有一个属性: `.value`

适用场景
基本类型: string, number, boolean, null, undefined
对象/数组: 也可以包裹对象，Vue 会自动将其转换为 reactive

### reactive
reactive 接收一个普通对象，返回该对象的响应式代理（Proxy）

适用场景
复杂的嵌套对象
当你希望像操作普通对象一样操作数据，不想到处写 `.value` 时

### Options API（Vue 2风格）

```javascript
export default {
  data() {
    return {
      message: 'Hello Vue!',
      count: 0
    }
  },
  methods: {
    increment() {
      this.count++
    }
  }
}
```

### Composition API（Vue 3推荐）

```javascript
import { ref, reactive } from 'vue'

export default {
  setup() {
    // 基本类型使用ref
    const count = ref(0)
    const message = ref('Hello Vue!')
    
    // 对象使用reactive
    const state = reactive({
      name: 'Tom',
      age: 25
    })
    
    // 方法
    const increment = () => {
      count.value++
    }
    
    return {
      count,
      message,
      state,
      increment
    }
  }
}
```

## 模板语法

```html
<template>
  <!-- 文本插值 -->
  <h1>{{ message }}</h1>
  
  <!-- 属性绑定 -->
  <a v-bind:href="url">Link</a>
  <!-- 简写 -->
  <a :href="url">Link</a>
  
  <!-- 双向数据绑定 -->
  <input v-model="message" placeholder="edit me">
  <p>Message is: {{ message }}</p>
  
  <!-- 事件处理 -->
  <button v-on:click="handleClick">点击</button>
  <!-- 简写 -->
  <button @click="handleClick">点击</button>
  
  <!-- 条件渲染 -->
  <p v-if="show">显示内容</p>
  <p v-else>其他内容</p>
  
  <!-- 列表渲染 -->
  <ul>
    <li v-for="item in items" :key="item.id">
      {{ item.name }}
    </li>
  </ul>
</template>
```

## 计算属性
计算属性（Computed Properties） 是一种基于响应式依赖进行缓存的值。
当依赖的数据发生变化时，就会重新计算。目的是为了解决模板中写复杂的表达式

```javascript
import { computed, watch } from 'vue'

export default {
  setup() {
    const count = ref(1)
    
    // 计算属性
    const tripleCount = computed(() => count.value * 3)

    return { count, tripleCount }
  }
}
```

## 组件注册
组件是一个可复用的 Vue 实例，具有自己的模板（HTML结构）、逻辑（JavaScript）和样式(CSS)

### 子组件
```vue
<template>
  <div class="box">
    <h3>子组件</h3>
    <p>{{ msg }}</p>
    <button @click="handleClick">点击通知父组件</button>
  </div>
</template>

<script>
import { defineProps, defineEmits } from 'vue';

export default defineComponent({
  // 1. 定义接收的 Props
  props: {
    msg: String
  },
  // 2. 定义发出的事件
  emits: ['my-event'],
  setup(props, { emit }) {
    function handleClick() {
      emit('my-event', '来自子组件的数据')
    }
    return {
      handleClick
    }
  }
})
</script>

<style scoped>
.box { border: 1px solid #ccc; padding: 10px; margin-top: 10px; }
</style>
```

### 父组件

```vue
<template>
  <div id="app">
    <h1>父组件</h1>
    
    <!-- 模板中使用变量和方法，与 script setup 完全一致 -->
    <Child 
      msg="Hello Vue3" 
      @my-event="handleReceive" 
    />
    
    <p v-if="receivedData">父组件收到: {{ receivedData }}</p>
  </div>
</template>

<script>
import { ref } from 'vue';
import Child from './Child.vue';

export default {
  name: 'App',

  // 1. 注册组件（依然需要显式注册，除非使用全局注册或插件）
  components: {
    Child
  },

  // 2. setup 函数是 Composition API 的入口
  setup() {
    // 3. 定义响应式数据
    const receivedData = ref('');

    // 4. 定义方法
    function handleReceive(data) {
      receivedData.value = data;
    }

    // 5. 【关键】必须将需要在模板中使用的变量和方法 return 出去
    return {
      receivedData,
      handleReceive
    };
  }
};
</script>
```
