## Spring Boot Redis

Spring Boot Redis最佳实践

### 依赖配置

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-redis</artifactId>
</dependency>
```

### application配置

```yaml
spring:
  redis:
    host: 127.0.0.1 # Redis服务器地址
    database: 0 # Redis数据库索引（默认为0）
    port: 6379 # Redis服务器连接端口
    password: # Redis服务器连接密码（默认为空）
    timeout: 300ms # 连接超时时间（毫秒）
```

### 配置文件

```java
@Configuration
public class RedisConfig {

    @Bean
    public RedisTemplate<String, Object> redisTemplate(RedisConnectionFactory redisConnectionFactory) {
        RedisSerializer<Object> serializer = redisSerializer();
        RedisTemplate<String, Object> redisTemplate = new RedisTemplate<>();
        redisTemplate.setConnectionFactory(redisConnectionFactory);
        redisTemplate.setKeySerializer(new StringRedisSerializer());
        redisTemplate.setValueSerializer(serializer);
        redisTemplate.setHashKeySerializer(new StringRedisSerializer());
        redisTemplate.setHashValueSerializer(serializer);
        redisTemplate.afterPropertiesSet();
        return redisTemplate;
    }

    public RedisSerializer<Object> redisSerializer() {
        //创建JSON序列化器
        Jackson2JsonRedisSerializer<Object> serializer = new Jackson2JsonRedisSerializer<>(Object.class);
        ObjectMapper objectMapper = new ObjectMapper();
        objectMapper.setVisibility(PropertyAccessor.ALL, JsonAutoDetect.Visibility.ANY);
        //必须设置，否则无法将JSON转化为对象，会转化成Map类型
        objectMapper.activateDefaultTyping(LaissezFaireSubTypeValidator.instance,ObjectMapper.DefaultTyping.NON_FINAL);
        serializer.setObjectMapper(objectMapper);
        return serializer;
    }
}
```

## Spring Boot 缓存
原理是基于动态代理

### 依赖配置

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-cache</artifactId>
</dependency>
```

### 配置文件
Spring Boot入口类中加入`@EnableCaching`注解开启缓存功能

```java
@EnableCaching
@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class,args);
    }
}
```

### 注解
|注解|用途|
|---|---|
|@CacheConfig|公共配置。常用注解在class之上,统一指定value值|
|@Cacheable|主要方法返回值将被加入缓存。同时在查询时，会先从缓存中获取，若不存在才再发起对数据库的访问。|
|@CachePut|每次执行函数并将结果存入缓存，适用于update方法|
|@CacheEvict|清空指定缓存，适用于delete方法|
|@Caching|配置于函数上，组合多个Cache注解使用|

### 示例

```java
@CacheConfig(cacheNames = "student")
public interface StudentService {

    Student create(Student student);

    @Cacheable(key = "#p0")
    Student getById(Long id);

    @CacheEvict(key = "#p0")
    void deleteById(Long id);

    @CachePut(key = "#p0.id")
    Student updateById(Student student);
}
```

### 依赖配置

```xml
<dependency>
```

```xml
    <version>3.23.0</version>
</dependency>
```

### 初始化配置

```java
@Configuration
public class RedisConfiguration {

    /**
     * 创建 RedisTemplate Bean，使用 JSON 序列化方式
     */
    @Bean
    public RedisTemplate<String, Object> redisTemplate(RedisConnectionFactory factory) {
        // 创建 RedisTemplate 对象
        RedisTemplate<String, Object> template = new RedisTemplate<>();
        // 设置 RedisConnection 工厂。它就是实现多种 Java Redis 客户端接入的秘密工厂。
        template.setConnectionFactory(factory);
        // 使用 String 序列化方式，序列化 KEY 。
        template.setKeySerializer(RedisSerializer.string());
        template.setHashKeySerializer(RedisSerializer.string());
        // 使用 JSON 序列化方式（库是 Jackson ），序列化 VALUE 。
        template.setValueSerializer(RedisSerializer.json());
        template.setHashValueSerializer(RedisSerializer.json());
        return template;
    }

    @Bean
    public RedissonClient redissonClient() {
        Config config = new Config();
        config.useSingleServer().setAddress("redis://127.0.0.1:6379").setPassword("password");
        // 根据config创建出RedissonClient实例
        return Redisson.create(config);
    }
}
```

### 示例

```java
@Slf4j
@RestController
public class SecKillController {

    @Resource
    private RedissonClient redissonClient;

    @Resource
    private RedisService redisService;

    @GetMapping("/secKill")
    public String secKill() throws InterruptedException {
        RLock lock = redissonClient.getLock("key:product");
        if (!lock.tryLock(5, TimeUnit.SECONDS)) {
            log.info("sorry");
            return "sorry";
        }
        try {
            //减库存
            int stock = (int) redisService.get("product:0");
            if (stock <= 0) {
                log.info("stock is empty");
                return "stock is empty";
            }
            Long decr = redisService.decr("product:0", 1);
            log.info("扣减成功，剩余库存:" + decr);
        } catch (Exception e) {
            log.error("", e);
        } finally {
            lock.unlock();
        }
        return "success";
    }

    @GetMapping("/fill")
    public String fill() {
        redisService.set("product:0", 100);
        return "success";
    }

}
```
